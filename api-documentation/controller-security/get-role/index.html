        current version:

        getRole

        Since Kuzzle v1.0.0

        HTTPOther protocols

        URL: http://kuzzle:7512/roles/<role id>
        Method: GET

        Query

        {
  "controller": "security",
  "action": "getRole",

  "_id": "<roleId>"
}

        Response

        {
  "status": 200,                     
  "error": null,                     
  "result": {
    "_id": "<roleId>",                // The role id
    "_source": {
      "controllers": {
        ...
      }
    }
  },
  "index": "%kuzzle",
  "collection": "roles"
  "action": "getRole",
  "controller": "security",
  "volatile": {},
  "requestId": "<unique request identifier>"
}

        Given a role id, retrieves the corresponding role from the database.

        The role unique identifier. It's the same you set when you create a role. in its responses when you create a role.
